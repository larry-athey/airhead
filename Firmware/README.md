# Updating Your Firmware

This firmware updater is only for Windows. If you are a Mac or Linux user, you have two options.

1. Install the Arduino IDE and all required libraries
2. Install Windows in a local virtual machine such as VirtualBox or VMware

No special equipment is needed beyond a USB-C data cable _(not a charging-only cable)_. This would be the same cable that you'd use to transfer photos from your phone to your computer.
