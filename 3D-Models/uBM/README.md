# μBoilermaker (micro Boilermaker) 

The μBoilermaker is 100% based on the Airhead, but is a stand-alone boiler controller for 120 volt resistive loads up to 1500 watts and uses PID temperature control, unlike the aggressive PI controller used by the Airhead. You can switch any Airhead into μBoilermaker mode by holding the Value+ buttong for 15 seconds after you have selected Mode 3. Use the same procedure to switch back.
