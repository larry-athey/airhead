# μBoilermaker (micro Boilermaker) 

The μBoilermaker is 100% based on the Airhead, but is a stand-alone boiler controller for 120 volt resistive loads up to 1500 watts and uses PID temperature control, unlike the aggressive PI controller used by the Airhead. You can switch any Airhead into μBoilermaker mode by holding the Value+ buttong for 15 seconds after you have selected Mode 3. Use the same procedure to switch things back.

The reason for the μBoilermaker is to satisfy the requests for a Bailermaker that isn't WiFi dependent. This isn't a one-to-one implementation of the full size Boilermaker, but it provides a person the ability to have constant power, constant temperature, or timed runs with progressive temperature. While there isn't a **Cruise Then Brew** mode, you can accomplish the same thing manually.
